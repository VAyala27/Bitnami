Deploy from a Private Helm and Docker Registry Through Kubeapps

The new release of Kubeapps now supports the deployment from private Helm repositories with private Docker images. Now you can associate a Docker image pull secret to an application repository from the Kubeapps UI to simplify the access to private repos within a Kubernetes RBAC authentication model. Both Kubernetes cluster operators and users will benefit of this new feature, so check out the latest feature now.
